# AIx4

Compare AI chat responses from ChatGPT, Claude, Gemini, and Grok side-by-side in a single desktop app.

| Windows | [Download .exe](https://github.com/kylesamani/aix4/releases/latest/download/AIx4-Setup-1.1.1.exe) |

The app checks for updates automatically on launch and can also be triggered via **Help > Check for Updates**.

## Features

- Send the same prompt to all four AIs simultaneously
- Grid, column, and focus layout modes
- Drag-and-drop to reorder AI panels
- Tab support for multiple conversations
- Compare responses with Claude API analysis
- Share conversation links
- Light/dark/system theme
- Auto-updates from GitHub Releases
